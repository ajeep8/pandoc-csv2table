Grid Table

It lets you inlcude block markdown as well, unlike multiline tables. However, it does not support column alignments. Every column is default-aligned.

Final Words

You only need either a multiline table or a grid table. Multiline tables allow control over column alignments. Grid tables allow inclusion of block level elements.
